Программирование на языке JavaScript

С использованием HTML, CSS и JavaScript разработан простой калькулятор.
Калькулятор выполняет основные арифметические действия, такие как сложение, вычитание, умножение и деление. Работает с десятичными и отрицательными числами, имеет кнопку очистки экрана, кнопку backspace, которая удаляет последнюю введенную цифру. Имеет функцию памяти, позволяющую сохранять и вызывать число. Оформлен с помощью CSS. Реализовано регрессионное тестирование.
